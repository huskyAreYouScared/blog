# 使用自定义字体

  • 注意版权问题
  • 首先定义字体
  @font-face {
    font-family: 'selfFamily'; /* 名字任意取 */
    src: url('selfFont.woff2') format('woff2'),
         url('selfFont.woff') format('woff'),
         url('selfFont.ttf') format('truetype');
    font-style:normal;
    font-weight:normal;
  }
