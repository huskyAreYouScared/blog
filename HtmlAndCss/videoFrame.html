# 截取视频图片

TIP

本文要用到的知识: vue 、ElementUI 、es6相关知识

# 业务背景

  • 上传视频的时候，需要顺便上传一张视频封面，视频封面来自于视频中的某一帧

# 需求分析

  • 首先这个视频需要通过 input[type="file"] 表单元素选中要上传的视频，这里使用的是elementUI的上传组件
  • 通过上传组件的before-upload钩子函数来判断上传的内容是否为视频，如果是就要执行截取视频某一帧的逻辑
  • 截取视频某一帧图片，并上传到服务器，返回图片存在服务器的URL信息
  • 其余的按照正常上传逻辑走

  :accept="accept"
